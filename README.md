# Static

> 使用 Gitee 仓库作为静态资源库!

## 使用

- 生成 gitee [私人令牌](https://gitee.com/profile/personal_access_tokens)
- 创建静态资源仓库或使用已有仓库(最好 public,private 可以上传,无法直接访问文件)
- 配置仓库
- enjoy!
